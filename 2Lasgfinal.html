Stage "1" Brief Notes

~Not-So Brief Time Spent Putting This Together ;)

HTML-
is like the structure of a house.

CSS-
is like the style in which the house is decorated (carpet, paint color, etc.)

JAVAScript-
is like a remote control to the t.v. It controls the interactive elements in a file.

DOM-
is like a tree configuration that supports the structure of the HTML document.

ELEMENT-
helps to add structure to a document by identifying particular sections. Each element in most cases, contain opening and closing tags.

PARENT/CHILD ELEMENTS-
are formed when an element is included in, (has been preceded in the text by) another element. The former is called the "Parent", while the latter is called the "Child".

ATTRIBUTE-
selects the characteristic of an element to be addresses (usually preceded by a space and followed by "value".)

VALUE-
defines or gives a specfic descritionmto the attribute, such as "green","250 px", "bold"...etc.

*It is very import to try to be neat, clean and efficient when creating an HTML doc.

-This will help to make the proofreading and validation phase easier and quicker.

-Also, with less volume and less characters there are fewer oppportunities for errors to creep in.

-Plus, with more efficiency, your time spent will begin to produce more, higher quality projects.

**A few methods to achieve efficiency include:

-Combining reoccuring elements on stylesheets

-Correctly placing Parent/Child elements in an easy to follow indexed structure

-Using quality validation tools such as the W3C website.
